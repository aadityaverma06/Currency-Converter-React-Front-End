# Currency Converter App

A simple Front-End app that can convert currencies. 

## ✅ Features 

- Easy to use.
- Convert to/from any currency.
- Swap between the selected currencies with a simple click.
- Dynamic Currency Data.

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- API Call

## Installation / Access
### 📦 On Local Machine
```bash
npm run dev
```

### 🌐 Using Vercel Link
